# Vibora

VIBORA – The Mood Engine is a mood based recommendation web app built with HTML, CSS, and JavaScript. It suggests what to watch, listen to, eat, drink, do, and read based on how you feel, featuring a clean UI with Google Fonts and UIverse components.

# Features 

 - Mood Selection System
 - Watch Recommendations
 - Music Suggestions
 - Food & Drink Ideas
 - Book Recommendations
 - Activity Suggestions
 - Clean, Responsive UI
 - Engaging animations and interactive components
 - Lightweight and fast performance

# Tech Stack

 - HTML5 – Structure
 - CSS3 – Styling & Layout
 - JavaScript (Vanilla JS) – Logic & Interactivity
 - Google Fonts – Typography
 - UIverse CSS Library – Enhanced UI components
